## 🏗️ How I Engineer (7 Steps)

1. **Read the rulebook** — extract constraints & objectives  
2. **Model fast** — CAD solid models & rough kinematic sketches  
3. **Analyze** — FEA, hand calcs, tip‑over angles, FoS checks  
4. **Prototype / fixture** — CNC, weld, 3D-print, jig design  
5. **Instrument & test** — sensors, data logging, validation runs  
6. **Iterate** — trim weight, cost, and complexity; document changes  
7. **Deliver** — clean reports, BoMs, and one great hero photo
